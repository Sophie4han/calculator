# calculator

## 소개
Swift 기초 주차에 진행한 계산기 만들기 과제.
기본적인 사칙연산부터 클래스 간 역할 분리를 통해 확장성 고려하여 구현.

- 더하기, 빼기, 곱하기, 나누기 기능 구현
- 인스턴스 생성 후 연산 실행 및 결과 출력
- 추가적으로 연산(%) 기능 추가
